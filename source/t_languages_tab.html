Languages Tab

Settings for the Languages tab in the ABBYY FineReader Options window.

To access the Options window, select Tools > Options....

  1. In the Specify OCR languages manually box, specify these languages: English; French; German; Greek; Simple math formulas. If working on eb03, see OCR for eb03.

  2. Click OK when finished, to save all settings in the Options window.
Be alert for pages with other languages, like Hebrew and math formulas, that need different settings. Make a note of them in the inventory file. After you have output your full set of pages, go back to them and follow the procedure outlined in Math and Chemical Formulas to change the language settings for Hebrew and formulas; then re-recognize that one page. You will have to insert your footnote codes again after this step. Then output just the changed page and allow the new file to replace the existing one.
