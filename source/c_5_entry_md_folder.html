5-entry-md Folder

A collection of files containing metadata for each entry page.

In collaboration with the Metadata Research Institute at Drexel University, we are able to automatically create subject terms for each entry file. These index terms are generated by HIVE, which outputs its results as CSV files, one for each entry. These files are stored in the 5-entry-md folder for the corresponding print edition and section.
