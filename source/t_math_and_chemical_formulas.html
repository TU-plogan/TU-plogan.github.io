Math and Chemical Formulas

Use custom language setting for math and chemical formulas, as well as Hebrew.

We always include formulas in our output. Unfortunately, ABBYY FineReader is not very good at capturing them accurately and is particularly poor with multiple-line formulas.

AFR does have two special languages that improve results when capturing formulas: Simple chemical formulas and Simple math formulas. To use either of these languages, or to add the Hebrew language to a page, follow the steps below.
  1. In AFR, open a page with one or more formula(s).
  2. In the main toolbar, select the Ë… icon next to the language window.
  3. In the drop-down menu, you may see the appropriate combination of languages in the list. Our normal combination is English; French; German; Greek; Latin. You will need to remove one language to include a formula language, so in the example above, the second combination is a good choice if there are not a lot of Latin phrases on the page.
  4. If the combination you need does not appear, select More languages....
  5. This will open the Language Editor window (above). There, you can select any combination of languages. Both Simple math formulas and Simple chemical formulas appear in the Formal Languages section at the bottom of the list.
  6. Return to the main window.
  7. Select Recognize page to redo the OCR.
  8. Add in the footnote @ symbols, if necessary.
  9. Select Save Page. In the dialog window that opens, navigate to the same folder where the rest of the HTML files for the section are stored and overwrite the existing file.
