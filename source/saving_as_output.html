Saving as Output

How to output your OCR results.

AFR can output results as anything from a Word document to a text or PDF file. We output as HTML because it is the easiest format to convert into TEI-XML.

  1. To output your finished pages, select Formatted text from the drop-down box on the main toolbar. Then choose Save.
  2. In the Save document as window, make sure you are saving as an HTML Document (*.htm) and select the option to Create a separate file. (We want separate files for each page.)
  3. Save to the appropriate 2-page-html folder, as explained in the section on the Print Edition Folder .
