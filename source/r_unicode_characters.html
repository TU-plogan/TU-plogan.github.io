Unicode Characters

List of unicode characters and entities used frequently in the Encyclopedia and not on the standard US keyboard.

  1. Some characters, like the middle dot, have multiple possibilities. For consistency, always use the one indicated here.
  2. If you cannot find a character, switch fonts to Arial Unicode MS. Times New Roman has < 3,000 characters; Arial Unicode MS has 38,917.
  3. For the complete table of unicode characters, see FileFormat.Info. For character entities, see Freeformatter.com.
  4. In addition to the Symbols window in ABBYY FineReader, you can also use the Windows Character Map to copy and paste special characters into AFR
    1. Type character map into the Windows search bar to locate the application, or use the Start menu and navigate to Windows Accessories > Character Map.
