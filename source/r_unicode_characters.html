Unicode Characters

List of unicode characters and entities used frequently in the Encyclopedia

  1. Some characters, like the middle dot, have multiple possibilities. For consistency, always use the one indicated here.
  2. Use Arial Unicode MS whenever you cannot find a character. Times New Roman has < 3,000 characters; Arial Unicode MS has 38,917.
  3. For the complete table of unicode characters, see FileFormat.Info.
  4. For character entities, see Freeformatter.com.
