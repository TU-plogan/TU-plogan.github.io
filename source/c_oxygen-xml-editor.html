Working with XML

This introduction to Oxygen XML Editor shows you how to navigate the interface and perform standard procedures on the Encyclopedia files.

After finishing the OCR process, we switch to Oxygen as our workhorse. We use it to transform our HTML files into TEI, to proofread and edit them, and to validate everything. Oxygen is a powerful tool for any work with XML. It is easy to learn and has a user-friendly interface, similar to a WISIWYG editor. Knowing the basics of TEI when working with files using that encoding schema will save a lot of frustration in the long run.

You can learn more about TEI at TEI Primer, from the Women Writers Project.

For further information about Oxygen, see the online User Guide.
