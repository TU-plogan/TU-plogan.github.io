  • Data Organization

    With over 100,000 files, the Knowledge Project needs a clear means of organizing its data. We use specific naming conventions for all files and folders, to order the large quantity of material.

    • Archive Folder

      Used for long-term storage of image files

    • Digital Edition Folder

      Contains files connected with generating an edition or running analyses

    • Information Folder

      Contains information related to the project

    • Production Folder

      Contains all files involed in creating the textual data from page images of the Encyclopedia Britannica

Archive Folder

Used for long-term storage of image files

This folder is long-term storage for original image files of print editions of the Encyclopedia. We never modify these files. Instead, we copy select images to the production/images folder for active use.

The folder also includes scans of additional Encyclopedia Britannica editions, as well as alternate scans of production editions.

Once you have selected the images we need for current production, copy the files from eb-corpus/archive to the eb-corpus/production/images folder.
Note: Think of the eb-corpus/archive folder as a library, while eb-corpus/production/images is a bookcase with just the books we want to read.
eb03, eb07, eb09, eb11
These are the print edition folders. Each one contains all image files related to that edition. Each print edition folder includes two subfolders.
alternate
If there are multiple image sources for the edition, they are stored here.
ia-gri (name varies)
We keep our source images for production in a folder named with the online source that hosts it, using the following abbreviations.
bell bell reference
cdl california digital library
culc cornell university library collection
gb google books
gri getty research institute
ht hathi trust
ia internet archive
pg project gutenberg
uch university of chicago
other-editions
We store images from other editions of the EB here.
