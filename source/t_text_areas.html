Text Areas

How to draw text areas in .

  1. Select the Draw Text Area tool in the icons panel above the Image pane.
  2. For a simple two-column page, draw two text blocks around the body text, one for each column. Always begin with the left column, before drawing the right column. Ignore running headers, marginal call-outs, and page numbers.
  3. If the page contains footnotes, create a separate text block for the footnote region after completing both of the text blocks. If there are separate note regions at the bottom of both columns, creates separate text areas for each.
