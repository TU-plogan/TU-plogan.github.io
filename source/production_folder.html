Production Folder

>The production folder is one of two major organization units for the Encyclopedia Project. It is dedicated to the process of creating the TEI-XML master files. We then use those files to create new digital editions, in the digital-edition folder.

We start the review of data organization with the production folder and its subfolders, because it represents the first stage of the editing process. This is a complex folder and the one you will use most when working on the OCR process. The root folder contains subfolders for each edition, pages images, and scripts used to convert OCR output into the XML form we use for data preservation.

eb03, eb07, eb09, eb11
These are called the "print edition" folders. Each one contains all files related to that edition.
images
All original page images used in generating data for the four editions are copied from the archive folder and stored here.
python
We scan pages into separate files, but in the end we want files for each entry term in the encyclopedia, not each page. We use Python to generate new entry files from the original page files. We also use Python to reposition footnote text within the body of the text, rather than at the end of a page, following guidelines from the Text Encoding Initiative (TEI).
xslt
Our OCR program outputs data in a webpage format, as Hyper Text Markup Language (HTML). But for data storage, we want everything converted to eXtensible Markup Language (XML). We make this conversion use XSLT, a scripting language specifically for XML. We will use XSLT again in the digital edition folder, to convert XML into any number of common file formats, including HTML and PDF.
Note: The production folder also contains a PDF copy of this manual and production.xslx, a spreadsheet used to track the production process.
