ocr Folder

Contains all files involed in creating textual data from page images of the Encyclopedia Britannica.

The ocr folder is dedicated to the process of creating text from images. This folder and its subfolders represent the first stage of the editing process, and you will spend more time here than anywhere else when working on the OCR process. The root folder contains subfolders for each print edition and their pages images.

Each print edition folder contains an afr-user folder. This is where we save the custom dictionaries and user files for use with ABBYY FineReader. There is one set of user files for each edition. See OCR Tab and Areas and Text Tab.

eb03, eb07, eb09, eb11
These are called the "print edition" folders, because each folder contains all ocr-project files related to one edition, divided into sections. Once the OCR process is complete, the output files are saved to kp2/page/ebnn/snn/2-page-docx.
images
Original page images used in generating data for the four editions are copied from the archive folder and organized into sections that are stored here.
