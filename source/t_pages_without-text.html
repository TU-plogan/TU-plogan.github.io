Pages without Text

When to include pages without text in the output

All editions have blank pages and full-page illustrations with captions. Do you include them in the output or not? It depends on whether or not the pages are numbered.
  • If the page is numbered, then it has to be included in the output to keep the page numbering correct. You do not need to draw any boxes, as the program will automatically generate a blank page for it.
  • If the page is not numbered, but instead is identified as a "Plate," it should be deleted from the OCR project. These files are supposed to be removed before the image files are organized into the Create an Image Collection, so you should not encounter many of them.
