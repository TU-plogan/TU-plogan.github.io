Section Folder

Each print edition section folder (r01, r02, r03, … ) contains subfolders for the different types of production files we generate, preceded by a numeral that reflects their position in the production sequence.

1-afr-project

These are the AFR “OCR Project” folders used to organize the 250-page section. OCR project folders retain the text boxes and language settings, and can be used to output text in multiple formats. The folder name includes the edition, section, and the afr tag (eb09-r01-afr). New OCR project folders overwrite earlier ones, so we only keep the most recent version.
Important: OCR project folders contain hidden files by default that are essential to their retention of text boxes, etc. When creating an OCR project folder, change the attribute value on the three hidden files to unhidden to ensure that they are included whenever the folder is copied. For more information, see Create an OCR Project Folder.

2-page-html

This folder contains the HTML files created by AFR, with one file per print page (250 files). Later in the process, we generate separate files for each entry, rather than each page, so we use folder names to differentiate the page files from the entry files. HTML file names use the convention: edition, section, page-number (1-250). Thus the file eb09-r01-0005.htm contains the text of the fifth page in the first 250-page section of the letter "R" for the ninth edition.
Note: Best Practice: AFR 14 adds spaces within the filenames that we do not want. It also creates a folder containing a single CSS file for each HTML file, appending _files to the same root filename. Finally, it sets up a link between the HTML file and the new folder, so that if you move one, it also moves the other.

To break the link and correct the file names, use a program like Bulk Rename Utility. Select all of the HTML files and rename. In the figure below, AFR added - nnnn to each file name. It was corrected using the options in Replace (3). Preview results in the New Name column. Use the Rename button (not shown) to implement the changes.

After renaming the files, you can move all of the CSS folders into a separate subfolder without disturbing the HTML files. Name the folder css.

3-page-tei

The HTML files are transformed into TEI-XML using an XSLT script. These are still single-page files, with the same name as the HTML files but a different files extension.
