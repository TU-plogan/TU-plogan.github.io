Data Organization

A publication as large as this one needs a clear means of organizing its data. The Encyclopedia Project uses specific naming conventions for all files and folders, to manage its tens of thousands of files.

All files are saved in the eb-corpus folder on the DSC server.

Each of the folders holds files for different aspects of the project and the process of creating new digital editions of the historic encyclopedias.

  1. Archive Folder
    This is our long-term storage for original image files of print editions of the Encyclopedia. We never modify these files. Instead, we copy them to the production/images folder for active use.
  2. Digital Edition Folder
    The second stage in the project is creating editions that we can analyze or place on the web. All files connected with either generating an edition or running an analysis of its data are contained here.
  3. Information Folder
    We use this folder to organize useful information on TEI, OCR, metadata, the history of Encyclopedia Britannica, and more, including this manual.
  4. Production Folder
    The first stage of the project is creating the textual data from page images of the Encyclopedia. All files in this initial process are contained here.
