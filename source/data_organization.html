Data Organization

A publication as large as this one needs a clear means of organizing its data. The Encyclopedia Project uses specific naming conventions for all files and folders, to manage tens of thousands of files.

All files are saved in the eb-project folder on the DSC server.

Each of the folders holds files for different aspects of the project and the process of creating new digital editions of the historic encyclopedias.

archive
This is our long-term storage for original image files of print editions of eb. We never modify these files. Instead, we copy them to the production/images folder for active use.
digital-edition
The second stage in the project is creating editions that can analyzed or placed on the web. All files connected with either generating an edition or running an analysis of its data are contained here.
information
Useful information will be found here on TEI, OCR, project meta data, the history of eb, and more (including this manual).
production
The first stage of the project is creating the text dataset from page images of eb. All files in this initial process are contained here.
