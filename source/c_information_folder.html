Information Folder

We use this folder to organize useful information on TEI, OCR, metadata, the history of Encyclopedia Britannica, and more, including this manual.

This includes all documentation on processes for working with the Nineteenth-Century Knowledge Project — including this document — publication and content data about the different editions of the encyclopedias, settings for Oxygen XML Editor, guidelines for using ABBYY FineReader, and so forth. Anything that is in the nature of general information about tools or techniques we use will be saved here.
