Format Settings Tab

Settings for the Format Settings tab in the ABBYY FineReader Options window.

To access the Options windown, select Tools > Options....

  1. Select DOC(X) / RTF / ODT as the Document type. The Document layout should be Formatted text.
  2. Leave "Default paper size" as Standard.
  3. Everything else should be left unchecked.

  4. Click OK when finished, to save all settings in the Options window.
