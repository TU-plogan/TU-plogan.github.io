Create a Transformation Scenario

How to locate the appropriate XSLT file and create a new transformation scenario.

About this task

If you do not see eb-html2tei, you will have to locate it.

Procedure

  1. Select New > XML Transformation with XSLT. This opens the New scenario window. New scenario is the same as Edit scenario but with less pre-populated information.
  2. Name the new scenario eb-html2tei.
  3. On the XSLT tab, click the folder icon for the XSL URL field and navigate to eb-corpus/production/xslt/. Select eb-html2tei.xsl.
  4. Continue with the Step 3 in Transform the HTML Files to TEI.
