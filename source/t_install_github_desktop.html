Install Github Desktop

Set up GitHub to create the kp2 repository on your machine.

GitHub is a hosting platform that we use to synchronize kp2 files among different users. For a good introduction to basic GitHub concepts, like branches, commits, and pull requests, review the Hello World tutorial.

  1. Download and install GitHub Desktop.
  2. Clone the kp2 repository to your computer.
    1. In github Desktop, open the File menu, select Clone repository ....
    2. In the list that appears on the GitHub.com tab, select pm1951/kp2.
    3. Tell it where to save the files: under Local path, add D:\knowledge-project.
    4. Click Clone.
  3. Add GitHub to the taskbar, for ease of access.
    1. On the Windows Start menu, open the GitHub, Inc. folder, right-click GitHub Desktop, and select More > Pin to taskbar.
