Production Folder

The first stage of the Nineteenth-Century Knowledge Project is creating the textual data from page images of the Encyclopedia Britannica. All files in this initial process are contained here.

The production folder is one of two major organization units for the Knowledge Project. It is dedicated to the process of creating the TEI master files. We then use those files to create new digital editions, in the digital-edition folder.

The production folder and its subfolders,represent the first stage of the editing process. This is a complex folder and the one you will use most when working on the OCR process. The root folder contains subfolders for each print edition, pages images, and XSLT scripts used to convert OCR output into the XML form we use for data preservation.

eb03, eb07, eb09, eb11
These are called the "print edition" folders. Each one contains all files related to that edition.
images
All original page images used in generating data for the four editions are copied from the archive folder and stored here.
Python
We scan pages into separate files, but in the end we want to create files for each entry term in the encyclopedia, not each page. We use Python to generate new entry files from the original page files. We also use Python to reposition footnote text within the body of the text, rather than at the end of a page, following guidelines from the TEI.
xslt
Our OCR program outputs data in a webpage format, as HTML. But for data storage, we want everything converted to XML. We make this conversion use XSLT, a scripting language specifically for XML. We will use XSLT again in the digital edition folder, to convert XML into any number of common file formats, including HTML and PDF.
Note: The production folder also contains production.xslx, a spreadsheet used to track the production process.
