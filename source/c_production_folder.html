Production Folder

Contains all files involed in creating the textual data from page images of the Encyclopedia Britannica

The production folder is one of two major organization units for the Nineteenth-Century Knowledge Project. It is dedicated to the process of creating the TEI master files. We then use those files to create new digital editions, in the digital-edition folder.

The production folder and its subfolders,represent the first stage of the editing process. This is a complex folder and the one you will use most when working on the OCR process. The root folder contains subfolders for each print edition and their pages images.

eb03, eb07, eb09, eb11
These are called the "print edition" folders. Each one contains all files related to that edition.
images
All original page images used in generating data for the four editions are copied from the archive folder and stored here.
Note: The production folder also contains two spreadsheets. The first, print-pagination.xslx, records the page spans for each letter in the different editions of the EB. The second, production.xslx, documents the exact contents of each section we create in the production process.
