2-page-html Folder

Container for HTML page files created by ABBYY FineReader.

This folder contains the HTML files created by ABBYY FineReader, with one file per print page (250 files). Later in the process, we generate separate files for each entry, rather than each page, so we use folder names to differentiate the page files from the page files. HTML file names use the convention: edition, section, page-number (1-250). Thus the file eb09-r01-0005.htm contains the text of the fifth page in the first 250-page section of the letter "R" for the ninth edition.
