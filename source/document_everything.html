Document Everything

All activity on the project needs to be properly documented.

The Encyclopedia project contains tens of thousands of individual files, including page images, AFR project files, HTML output files, derivative TEI files, and more. It is a multi-year project involving many researchers over time. In addition to a careful scheme of data organization, we maintain careful records of what each operator did (or did not do) and when. This ensures that nothing is lost when we transition between researchers, and that we have the documents when need if, later, we need to go back to unravel some mystery.

The primary documentation for researchers involved in the OCR stage is the inventory file, a spreadsheet created for every edition section and its AFR Project folder. The inventory file documents the production history of that section.

  1. Open a new XSLX (Excel) file and save it as yyyymmdd-ebnn-rnn-inventory in the home directory for the edition and section it documents.
  2. Create column labels as follows, or copy them from an existing inventory spreadsheet:
  3. Enter the original filenames of all the images that make up the section.
    Excel will populate the list for you if you enter the first 2-4 names, select and drag your cursor down the column.
    The image filenames are the names of the images in the production/images directory.
  4. Enter the volume and corresponding printed page number for each image.
  5. The rest of the spreadsheet gets filled in as we output the OCR to HTML and review the results.
