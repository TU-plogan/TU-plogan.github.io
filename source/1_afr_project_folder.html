1-afr-project Folder

Container for OCR Project files for AFR.

These are the AFR “OCR Project” folders used to organize the 250-page section. OCR project folders retain the text boxes and language settings, and can be used to output text in multiple formats. The folder name includes the edition, section, and the afr tag (eb09-r01-afr). New OCR project folders overwrite earlier ones, so we only keep the most recent version.
Important: OCR project folders contain hidden files by default that are essential to their retention of text boxes, etc. When creating an OCR project folder, change the attribute value on the three hidden files to unhidden to ensure that they are included whenever the folder is copied.
