OCR

The procedures we use to get the best quality text recognition in ABBYY Fine Reader.

ABBYY FineReader is a complex program with many options for managing the OCR process. The following instructions are written for AFR 14, but the difference between it an the previous edition are minor, so this information should apply to both. The interface is defined for offices converting paper documents to PDF or Word files, and we will ignore most of that to focus on the OCR engine.

A good general introduction to the program is the AFR 14 User Guide online. On the first page, you will see a link to download it as a PDF, if you wish.
