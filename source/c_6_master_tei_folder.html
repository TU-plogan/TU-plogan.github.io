6-master-tei Folder

Contains the master files for creating digital editions.

Creating master files is the final goal of production in the Nineteenth-Century Knowledge Project. Each file combines the TEI entry file with its individual subject headings. We call them master files because we use them to generate different digital editions of the data in any desired format, such as html or TXT.
