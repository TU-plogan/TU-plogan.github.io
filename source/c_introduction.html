Introduction

What is the Encyclopedia Project and how does it work?

Objective

The Temple Encyclopedia Project is creating a standards-compliant data set of text from four historical editions of the Encyclopedia Britannica (eb), the third, seventh, ninth, and eleventh. While these editions are available on the web, their textual data is too inaccurate to be used for text mining. When completed, the Temple editions will become freely available to researchers, making it possible to arrive at a better understanding of how knowledge was socially constructed in the nineteenth century.

Production Method

The project begins with running OCR on page images of the Encyclopedia. The text is first output as single pages in HTML format, before being converted into XML. The data is encoding with the standards developed by the TEI. Python is used to combine the TEI pages into a single file and segmented into a separate file for each entry in the Encyclopedia. Finally, we add topical metadata to each entry.

These master files can then be used to generate editions in a variety of formats, from XHTML to EPUB to TXT. Finally, we run textual analysis routines on the data, so that we can better understand the structure of the Encyclopedia and how it changed during the course of the century.

Accessibility

When complete, the data set will include all text for approximately 100,000 entries. All of this material will be made freely available online through two public repositories, both in a user-friendly individual entry form, and in bulk download form for researchers.
