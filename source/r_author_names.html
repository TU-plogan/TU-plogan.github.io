Author Names

Encoding author initials at the end of entries.

Many entries end with the initials of the author. These initials are expanded elsewhere in the encyclopedia, and v1 includes expanding them in place, so users can identify the author.

  • Use <byline> for the initials. This is a freestanding tag that must be on its own line and cannot be enclosed in a <p> tag.
    • Do not include <docAuthor> within <byline>, as that is reserved for the author of an entire document, not an author for a subset of a document, like an entry.
    • <byline> is part of the TEI <closing> group, so it has to be at the end of a <div>. For long entries, where sections are written by different authors, enclose each authored section within <div type="section">.
  • For author names in citations within an entry, use <bibl><author>.
