Ordering Areas

How to order areas correctly for text output.

If text areas are ordered incorrectly, ABBYY FineReader will output sections of text or entire text columns in reverse order, making it unreadable.

  1. The order of the areas determines the order in which AFR will output the text on the page. It remembers the order in which they were drawn, but it also has a way to reorder the blocks at any time. To check or change the order, use the Reorder Areas tool in the icons panel. This causes the order number to appear in the upper-left corner of each block.
  2. Click in any text area to change it to #1, #2, and so forth. The other blocks are automatically reordered.
