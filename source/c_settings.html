Settings

Recommended settings for all options in ABBYY FineReader

The key to getting consistent, high-quality results with ABBYY FineReader is its many options for fine-tuning the OCR process. These are located on seven tabs in the Options window, accessed by selecting Tools > Options....

Settings for some options are specific to the edition in use. When ever you begin working on a different edition, always update the user file settings on the OCR Tab and the dictionary file settings on the Areas and Text Tab. For eb03, also check the instructions for OCR for eb03.

Note: To avoid losing settings on each tab, click OK when finished. This saves all settings in the Options window.
