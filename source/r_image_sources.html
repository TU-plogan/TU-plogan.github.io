Image Sources

Bibliographic information on image sources.

Each edition has a different image source. Bibliographical information for the print edition is given, followed by the location of the print source, if known. The digital archive where the image files are stored is given next with their file format. Finally, we acknowledge the organizations who digitized the print volumes.

Where images are missing or damaged, we replace them with an alternate source of the same edition. These are listed as "exceptions".

vols. 1-18
