Introduction

What is the Encyclopedia Project and how does it work?

Objective

The Temple Encyclopedia Project is creating a standards-compliant dataset of text from four historical editions of the Encyclopedia Britannica (eb), the 3rd, 7th, 9th, and 11th editions. While these editions are available on the web, their textual data is too inaccurate to be used for text mining. When completed, the Temple editions will become freely available to researchers, making it possible to arrive at a better understanding of how knowledge was socially constructed in the nineteenth century.

Production Method

The project begins with running OCR (Optical Character Recognition) on page images of the Encyclopedia. The text is first output as HTML of single pages, before being converted into XML with TEI encoding, the standard method for marking up textual data. Python is then used to combine the TEI pages into a single file and segmented into a separate file for each entry in the Encyclopedia. These become our master files and they are used to generate editions in a variety of formats, from HTML to PDF to TXT. They also are used to run various textual analysis routines on the data, so that we can better understand the structure of the Encyclopedia and how it changed during the course of the century.
