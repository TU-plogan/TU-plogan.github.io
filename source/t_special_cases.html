Special Cases

Situations requiring a note in the page-inventory file.

You will encounter a variety of situations that need to be brought to the attention of the person who converts page files to entry files. If you notice any of the following cases on a page, please check the appropriate column or add a note in the inventory file.

  • Blocks of poetry or drama. These require special coding in the TEI page file.
  • A table that runs over onto the next page. We will unite it into a single, continuous table in the entry file
  • Footnote text that runs over onto the next page. We will combine the text into a single note in the entry file.
  • Illegible or missing text. This also gets special coding in the TEI page file.
  • Multi-line formulas or equations. See Formulas and Equations.
  • Unusually complex tables that will need correcting later in the TEI file.
  • Uses of Greek or Hebrew that are too extensive to correct in the OCR process.
