Areas and Text Tab

Settings for the Areas and Text tab in the ABBYY FineReader Options window.

Before you begin

To access the Options windown, select Tools > Options....

Procedure

  1. Check all options for Verification settings.

  2. ABBYY FineReader needs to know where the shared user dictionary is located. Open the User dictionaries... window. Select English (United States) and use the Browse... button to navigate to the afr-user/UserDictionaries folder for the edition you are working on. for example, if you are scanning eb11, the dictionaries are located in production/eb11/afr-user/UserDictionaries.

  3. Click OK when finished, to save all settings in the Options window.
