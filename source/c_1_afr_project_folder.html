1-afr-project Folder

Container for OCR Project files for ABBYY FineReader.

ABBYY FineReader has a proprietary compressed-file folder structure for its OCR Projects, which we use to organize each 150- to 250-page section. The special data files in the OCR-Project retain the text boxes and language settings and can be used to output text in multiple formats. The folder name includes the edition and section name (eb09-r01).
DANGER: OCR-Project folders contain hidden files by default that are essential to their retention of text boxes, etc. When creating an OCR-Project, change the attribute value on the three hidden files to unhidden to ensure that they are included whenever the folder is copied.
