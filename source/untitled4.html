Check Entry Terms List

Compare our list of recognized entry terms with the print pages.

About this task

Python generates a list of all the entry terms it recognized. It will miss terms that may have been incorrectly OCR-ed, and it will truncate others with incorrect characters. We proofread the list of entry terms by comparing it directly to the original pages of the print edition, using either hard copies or PDF versions. All corrections must be noted in the Entries Inventory spreadsheet.
