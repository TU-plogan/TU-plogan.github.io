Edition-Section System

File organization depends on two basic folder types

We are digitizing four different editions of the Encyclopedia Britannica. We organize all text initially by the print edition number.

The original editions are organized alphabetically, so we further subdivide each edition by alphabetical letter. The full text of some letters is enormous in size, so we segment each letter by a numbered section of the letter.

This edition-section system gives us manageable chunks to work with from the first stage, the OCR process, all the way through to the TEI master files.

Most folders in the processing sequence are subdivided by this edition-section stystem.
