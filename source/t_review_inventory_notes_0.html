Review Inventory Comments

Address any issues raised in the page-inventory notes.

  1. Make any highlighted corrections mentioned in the TEI-P inventory file as saved for TEI-E (like fixing tables the runover to a second page). To find the correct TEI-E file, search the image file name.
  2. For tables that continue onto the next page, retain the <pb> but restructure the table as a continuous one, without repeating the table headers used by the printers.
  3. Cut all text for any partial entries at the start of the section and append it to the last entry of the previous section. Remove the partial entry and correct the filename for any other entries on the first page of the section.
  4. Correct any entry terms for new letters of the alphabet, since they have to be altered for Python to recognize them as an entry.
