Notes

Note text is always moved from its position on the print page and placed inline with the page text at the point where the note number is originally located. Note text is surrounded with the <note> element. All notes must include @anchored, @place, @type, and @n.

<note anchored="true" place="bottom" type="authorial" n="3">
@anchored Does the page image show the exact place of reference? Permissible values: true | false.
@place What was the original location of the note on the page image? Permissable values: bottom | margin
@type Who authored the note? In most cases, this will be the entry author, so use authorial." If the note is inserted by the Encyclopedia editors, use "editorial. Permissible values: authorial | editorial.
@n What is the note number? Sequence notes in each entry beginning with one.
