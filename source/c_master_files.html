Master Files

After creating entry files, we run an automated metadata generation process to add index terms to each entry.

Master files combine an entry file with metadata terms appropriate to the content, to assist in search and analytical operations. The folder and filenaming conventions match those of the entry folders and files.

To creating index terms for 113,000 entries, we use an automated procedure designed after extensive testing to achieve the highest relevancy results. This section describes the final process.
