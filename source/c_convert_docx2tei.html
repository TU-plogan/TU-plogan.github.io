Convert DOCX to TEI

Use Oxygen and XSLT to convert DOCX files to TEI.

ABBYY FineReader outputs DOCX files, but that is not a good storage medium for textual data. XML is far more flexible and makes it possible to output the data in many different files formats, including HTML, PDF, TXT, and CSV.

XML

XML has definite rules for how to structure data. It uses different schema that are tailored to different types of data. This allows users to select from any number of encoding schemes, or even create their own. These schemes have their own built-in sets of rules about what elements and attributes are legitimate.

TEI

We use the TEI XML schema because of its widespread acceptance for archival storage of textual data. This is a complex markup language that encodes the semanatic structure of a document, rather than its appearance. It allows us to define what is a title, subtitle, and body text. It does not tell us which typeface or font size to use. Later, users can display those elements in the way that best suits their purposes.

XSLT

To convert our HTML files to TEI, we use a "transformation" script written in XSLT. The script reads the DOCX files, strips out coding we do not want, renames the DOCX tags we do want into valid TEI tags, and generates a new TEI file for each of the HTML originals.
