Notes

Details method we use to code footnote text in TEI.

Note text is always moved from its position on the print page and placed inline with the page text at the point where the note anchor is originally located. Note text is surrounded with the <note> element. All notes must include @anchored, @place, @type, and @n.

Examples:

<note anchored="true" place="bottom" type="authorial" n="3">
<note anchored="false" place="margin" type="editorial" n="5">
@anchored Does the page image show the exact place of reference? Permissible values: true | false.
@place What was the original location of the note text on the page image? Permissable values: bottom | margin
@type Who authored the note text? In most cases, this will be the entry author, so use authorial. If the note is inserted by the Encyclopedia editors, indicated by [Ed.], use editorial. Permissible values: authorial | editorial.
@n What is the note number? Footnotes begin with the number one in each entry and are numbered sequentially in order of appearance. Marginal notes in the text are sometimes unnumbered, but we number them like all other notes.
