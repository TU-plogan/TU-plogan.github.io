metainfo Repository

The metainfo repository

The metainfo repository on Google Drive contains historical controlled vocabularies and other data needed to generate subject terms for inclusion in the TEI header of each entry file. It also include information about the HIVE vocabulary server at the Metadata Research Center at Drexel University.
