kp2 Repository

The folder structure of kp2.

The kp2 repository contains the post-OCR text files that are converted into master files, along with the digital editions derived from them. It is stored on GitHub and synchronized to computers in the Scholars Studio. The repository includes the following folders:

analytics Files used in analyzing all text in the Nineteenth-Century Knowledge Project.
code All scripts and programs used in the Knowledge Project.
digital-editions We use the master files to generate editions in different formats and store these derivative editions here.
entry TEI files with subject metadata in the header. We use these master files to generate derivative editions in multiple formats.
page Contains files at all stages of the process in going from the image files of original print pages to the master files.
