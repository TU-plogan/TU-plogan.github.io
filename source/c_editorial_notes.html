Editorial Notes

Check for notes by editors.

The 11th edition contains some footnotes written by the editors, rather than the author. Do a quick search to identify them and change @type.

  1. In Oxygen XML Editor use CTL + SHIFT + H to open the Find/Replace in Files window to search all files in the section.
  2. Add [ED.] to the Text to Find field and click Find All.
  3. Review the results window and select each result individually to open the file.
  4. Change the <note> @type value from authorial to editorial.
