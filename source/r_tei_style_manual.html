TEI Style Manual

TEI encoding practices for the Knowledge Project.

We use a subset of the TEI guidelines in creating the Nineteenth-Century Knowledge Project data set. Our goal is to create a base level of encoding for the v1 release (planned for late 2021), which can be expanded in v2 by adding finer levels of granularity.

Further information about TEI an be found at the following locations:

  • Comprehensive information about elements and attributes is available at P5: Guidelines for Electronic Text Encoding and Interchange.
  • For an introduction to TEI, see the Coding Tutorials from the Women Writers Project.
