Editing tables

Table columns and rows can be adjusted before recognition.

ABBYY FineReader has a small set of tools for adjusting tables to improve recognition accuracy.
  1. Click on the table to make the table tools menu bar appear.
  2. Use the pointer icon to grab and move row and column dividers with your mouse.
  3. Use the second icon to create a new row by clicking with the mouse where you want to position it.
  4. Use the third icon to create a new column.
  5. The yellow icons split and merge table cells.
