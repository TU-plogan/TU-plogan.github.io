Install Drive File Stream

Add Drive File Stream to the computer for accessing the kp1 repository.

Drive File Stream allows you to work directly on files stored in a Google Team Drive. These files remain in the cloud and do not take up space on the computer. However, when working on an OCR-Project, this constant downloading from the cloud slows down the operation too much, so we save a copy of the OCR folder to the hard drive.

You must be added to the Google Team Drive prior to installing the Google Drive File Stream software.

  1. Download and install Drive File Stream for Windows.
    This adds a virtual drive to the computer, called Google Drive File Stream. To work on files, open that drive in Explorer, just like before.
  2. To open kp1, go to Google Drive File Stream/Team Drives/kp1.
  3. Create a shortcut for kp1.
    1. Right-click on the folder icon, and select Send to > Desktop (Create a shortcut).
    2. Copy and paste the shortcut into the knowledge-project folder.
  4. Show the Drive File Stream icon in the taskbar.
    1. Place the cursor over a blankTaskbar Settings.
    2. Halfway down the page, click Select which icons appear on the taskbar.
    3. Change Google Drive File Stream from off to on.
  5. Change the cached files directory:
    1. Right-click the Drive File Stream icon in the taskbar, open Settings and select Open Preferences.
    2. Click Change next to Local cached files directory.
    3. Navigate to the D: drive and create a new folder called gdfs. Select that folder for the Local cached files directory.
  6. Download a local copy of the OCR folder to your computer:
    1. Open the Google Drive File Stream folder in Windows Explorer and go into the kp1 repository.
    2. Right-click the OCR folder and hover over the Drive File Stream icon. Select Available Offline.
