Set Image Resolution

Correct the image resolution on all pages of the OCR Project before drawing boxes.

The OCR process works best when the image files being scanned have of resolution of 300-600 dpi (dots-per-inch). Some of our source files have very high resolutions, and ABBYY FineReader will intterpret them as needing > 600 dpi resolutions, which causes problems for the scanner. So each time you create a new OCR-Project, follow this simple procedure.

DANGER
This process will erase any boxes, if you have already created them.
  1. With any page open in the Image Page, select the Edit Image icon.
  2. On the right-hand side of the Image Editor, open the Resolution options.
  3. If the resolution shown in Other is > 600 dpi, change it to 600 dpi.
  4. Set Selection to All Pages.
  5. Apply.
  6. Exit the Image Editor.
