Code Footnotes

How to using @ codes for footnotes.

About this task

In order to improve the accuracy of our encoding, we need to manually indicate all footnote references and note text in the Text pane. For this purpose, we use an easily-recognizable system of at codes.

Procedure

  1. In the body of the text, find the footnote reference number and insert @@ (2-@) before each.
  2. In the note blocks, where the footnote text is, insert @@@ (3-@) before each footnote number.
  3. Check that you have the same number of double and triple @'s before leaving the page.
