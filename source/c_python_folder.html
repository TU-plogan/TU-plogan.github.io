Python Folder

Python script files.

After our HTML files are transformed to TEI, we still have two critical changes to make to them before they can be used as digital edition masters. Both of these changes are done using a Python script.

  1. The footnotes in the HTML appear at the end or in the middle of the page, reflecting their position at the bottom of a column in the print edition. TEI is not print-based, so this format does not make sense. Instead, we want to move the note text into the body text at the point of attachment (i.e., where the original footnote reference appeared). Note text is surrounded by the <note> tag and can be displayed any way that we want in the digital edition.
  2. ABBYY FineReader outputs each individual page as a separate file. We need this format in order to create the original page numbers in the digital edition, but we do not want the edition to consist of individual pages. For the Encyclopedia, the entry is the basic unit of meaning, not page. We use Python to concatenate all of the page files into a single file, with pages correctly numbered. Python then separates the files every time it encounters a new entry title. In the end, we have a single file for each entry, with page breaks and original page number clearly marked.

These new page files are then ready to become the master files for our digital editions, after they have been further edited and have metadata added. But all of that happens in the digital-edition folder, so we move these entry files there, to begin the next chapter in creating our digital edition.
