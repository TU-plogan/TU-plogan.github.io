Convert HTML to TEI

Using Oxygen and XSLT to convert HTML files to TEI.

ABBYY FineReader outputs HTML files, but that is not the best storage medium for textual data. XML is far more flexible and makes it possible to output the data in many different files formats, including HTML, PDF, TXT, and CSV.

XML

While HTML has a set of specific rules for encoding text, XML has no rules of its own. Instead, it allows users to select from anyone of a number of encoding schemes, or even create their own. These schemes have their own built-in sets of rules about what elements and attributes are legitimate, for example.

TEI

We use the TEI scheme because of its widespread acceptance for archival storage of textual data. This is a complex scheme that encodes the structure of a document, rather than its display characteristics. It allows us to define what is a title, subtitle, and body text. It does not tell us which typeface or font size to use. Later, users can display those elements in the way that best suits their purposes.

XSLT

To convert our HTML files to TEI, we use a "transformation" script written in XSLT. The script reads the HTML files, strips out coding we do not want, renames the HTML tags we do want into valid TEI tags, and generates a new TEI file for each of the HTML originals.
