Import a Scenario

Learn how to import an existing xsl transformation scenario into an OCR Project.

The eb-docx2tei scenario is stored in the code directory and is easily imported into any OCR-Project.

  1. Open the Configure Transformation Scenario(s) window. .
  2. Select the Settings wheel in the upper-right of the window, and click on Import scenarios.


  3. In the dialogue box that opens, browse to eb-corpus/code/scenarios and select transformations.scenarios.


  4. The Import scenarios window opens. Select the transformation scenario(s) you want to import.


  5. If a copy already exists, go ahead and replace it.


  6. Continue with the steps in Transform the DOCX Files to TEI.
