Proof the Entry Terms List

Compare our list of recognized entry terms with the print pages.

Python generates a list of all the entry terms it recognized in processing a section. It will miss terms that may have been incorrectly identified, and it will truncate others with incorrect characters. We proofread the list of entry terms by comparing it directly to the original pages of the print edition, using either hard copies or PDF versions. All corrections must be noted in the comments column of the entry-inventory.

Note: Operators who are authorized to modify the entry files will want to do this with Oxygen XML Editor open, so you can correct entry terms in the files as you work, and can create new files for the missed entries.
