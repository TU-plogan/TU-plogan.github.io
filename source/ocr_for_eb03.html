OCR for eb03

Scanning eb03 requires special settings.

All the editions of eb have slightly different layouts, but additional considerations come into play when scanning eb03.

  1. Selection of languages. Most printed books from before 1820 use the "long-s" and eb03 is no exception. To address this problem, we add the long-s character to the character set for English and save the modified language file under a new name, like English-eb. We then add it to our list of specified OCR languages, on the Options > Languages tab.
  2. Selection of fonts. Caslon was one of the most popular typefaces of the eighteenth century, but Windows does not include it in its standard fonts, so AFR looks for the closest alternate. We get more accurate results when we install Caslon ourselves and then remove everything else from the list of fonts to use in the Options > OCR > Select fonts... window.
