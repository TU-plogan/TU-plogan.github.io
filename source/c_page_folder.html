page Folder

The container for text files used in processing OCR output into master files.

The page folder is dedicated to the process of modifying the initial OCR output to create the TEI master files. We then use those files to create new digital editions, in the digital-edition folder.

The page folder is structured similarly to the OCR Folder. It is subdivided into print edition folders and sections. Each section then contains folders for the different stages of processing each file. The numbering of the folders indicates the sequence of steps.
