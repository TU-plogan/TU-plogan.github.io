page Folder

Pages are individual printed pages in the Encyclopedia.

The page folder is dedicated to the process of converting the OCR output into TEI. These files are then further processed into the entry files that are stored in the entry folder.

The page folder is structured similarly to the ebnn Repositories. It is subdivided into print edition folders and sections. Each section then contains folders for the different stages of processing each file, as well as a copy of the page-inventory file for that section, named outputs-ebnn-snn.
