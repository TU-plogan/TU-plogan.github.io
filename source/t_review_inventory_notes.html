Review Inventory Comments

Correct resolvable issues noted in the comments

Comments by the ABBYY FineReader operator indicate issues on individual pages. Some of these are easily resolvable in the HTML files before conversion to TEI.

Note: Always indicate in the inventory file that the problem has been addressed, and whether it was addressed in the HTMl or TEI-page file.
  1. If there are footnotes that run over from one page to the the next, open both HTML pages. Find the runover footnote text in the second page. Copy it into the original note on the first page. Then delete it from the second page.
  2. If there are two or three references to the same note text on a page, you can duplicate the note text and add the necessary at codes for each note anchor to have its own note text. If there are more than two or three, this is impractible and needs to be save for the entry clean-up stage.
