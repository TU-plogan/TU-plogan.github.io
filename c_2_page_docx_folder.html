    2-page-docx Folder

    Container for Word .docx page files created by ABBYY FineReader.

    This folder contains the page files created by . Each file represents one printed page. All .docx filenames, like those below, use the convention: 
    edition - section - page-sequence-number
    The page-sequence-number is the sequence of the page in the section; pages in every section always begins at 0001, regardless of the printed page number.
    The file eb11-w03-0005.docx contains the text of the fifth page in the third section of the letter "W" for the eleventh edition.
